# Bright-Scholar-Madrasah-Alumni-Association
Alumni Association Portal for Bright Scholar Madrasah

## 🚀 Live Demo
The application is live and deployed on Railway:
*   **Live App**: [BSMAA Alumni Connect Login](https://bsm-alumni.up.railway.app/login)

## 🛠️ Technology Stack
*   **Backend**: Java 21, Spring Boot 3.5.x, Spring Data JPA, Hibernate, BCrypt (Password Security)
*   **Frontend**: HTML5, Thymeleaf, TailwindCSS, Lucide Icons
*   **Database**: MySQL
*   **Deployment**: Docker, Railway
